# PHASE 2: Homepage Implementation

## Overview
Build the GROWNA homepage - a modern B2B marketing landing page.

---

## Sections Required

1. **Header/Navigation**
   - Logo (GROWNA text or placeholder)
   - Nav links: Services, About, Contact
   - CTA button: "Get Started"

2. **Hero Section**
   - Headline: Growth-focused value prop
   - Subheadline: Brief description of services
   - Primary CTA: "Start Growing"
   - Secondary CTA: "Learn More"
   - Visual: Abstract tech illustration or gradient

3. **Services Section**
   - Grid of 6 service cards:
     - AI Solutions
     - SaaS Products
     - ERP / Odoo
     - Systems Integration
     - Automation
     - Digital Transformation

4. **Why GROWNA Section**
   - Trust indicators
   - Key differentiators (3-4 points)

5. **CTA Section**
   - Call to action for consultation
   - Contact prompt

6. **Footer**
   - Logo
   - Quick links
   - Contact info placeholder

---

## Implementation Order

1. Create base UI components (`Button`, `Card`, `Container`)
2. Build layout components (`Header`, `Footer`)
3. Build each section in order
4. Assemble in `app/page.tsx`
5. Verify responsive design
6. Run build and fix any errors

---

## Design Notes

- Use dark theme with accent colors
- Ensure mobile responsiveness
- Add subtle animations with Framer Motion
- Keep copy placeholder until final content provided

---

## Acceptance Criteria

- [ ] Homepage renders without errors
- [ ] All 6 sections present
- [ ] Responsive on mobile/tablet/desktop
- [ ] Build passes (`npm run build`)
- [ ] No console errors
